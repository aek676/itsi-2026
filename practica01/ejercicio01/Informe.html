Ejercicio Guiado 1: Creación de un Flujo "Hola Mundo"

Para este primer ejercicio, se ha seguido la guía paso a paso. El objetivo era familiarizarse con la interfaz de n8n creando un nodo 'Start' y conectándolo a un nodo 'Set'. Dentro del nodo 'Set', se configuró un valor para generar un mensaje simple de "Hola Mundo". Finalmente, se ejecutó el flujo manualmente ("Execute Workflow") para verificar la salida JSON correcta, confirmando que el flujo funcionaba como se esperaba.

Ejercicio Guiado 2: Interacción con una API REST Pública

En este ejercicio, el objetivo fue interactuar con una API REST pública. Se utilizó el nodo 'HTTP Request' para realizar una petición GET a la API (en este caso, 'reqres.in') y obtener datos específicos de un usuario. La captura de pantalla muestra la configuración del nodo con la URL de la API y la respuesta JSON exitosa, demostrando cómo n8n puede consumir y mostrar datos de servicios externos.

Ejercicio Guiado 3: Transformación y Limpieza de Datos

Este ejercicio se centró en la manipulación de datos. Después de recibir información (posiblemente de la API del ejercicio anterior), se utilizó el nodo 'Set' para limpiar y transformar los datos. Como muestra la captura, se renombraron campos (ej. 'first_name' a 'nombre') y se eliminaron otros ('avatar', 'support') para dejar solo la información relevante y en el formato deseado.
